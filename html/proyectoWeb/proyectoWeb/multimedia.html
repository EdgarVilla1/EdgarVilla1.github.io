RESUELVE LAS SIGUIENTES PREGUNTAS PARA EVALUAR TU CONOCIMIENTO



    ¿Qué es una red de computadoras?


    Un conjunto de computadoras conectadas entre sí
    Un programa informático
    Un dispositivo de almacenamiento

    ---------------------------


    ¿Cuál es el propósito principal de una red de computadoras?


    Compartir recursos y datos
    Jugar videojuegos en línea
    Navegar por internet

    ---------------------------


    ¿Qué es un protocolo de red?


    Un conjunto de reglas y estándares para la comunicación en una red
    Un tipo de computadora
    Un virus informático

    ---------------------------


    ¿Qué es un router?


    Un dispositivo que interconecta redes y enruta el tráfico
    Una persona que configura redes
    Un tipo de cable de red

    ---------------------------


    ¿Cuál es el estándar más común para las redes inalámbricas?


    Wi-Fi
    Bluetooth
    NFC

    ---------------------------


    ¿Qué es un firewall?


    Un sistema de seguridad que protege una red de amenazas externas
    Un dispositivo para conectarse a internet
    Un programa para enviar correos electrónicos

    ---------------------------


    ¿Cuál es la función principal de una dirección IP?


    Identificar de forma única un dispositivo en una red
    Enviar mensajes de correo electrónico
    Reproducir música en streaming

    ---------------------------


    ¿Qué significa el acrónimo LAN?


    Local Area Network (Red de Área Local).
    Large Area Network (Red de Área Amplia)
    Local Access Network (Red de Acceso Local)

    ---------------------------


    ¿Cuál es el objetivo principal del DHCP (Dynamic Host Configuration Protocol)?


    Asignar direcciones IP de manera automática
    Encriptar datos en una red
    Realizar copias de seguridad de archivos

    ---------------------------


    ¿Qué es un DNS (Domain Name System)?


    Un sistema que traduce nombres de dominio en direcciones IP
    Un tipo de virus informático
    Un software para diseñar sitios web

    ---------------------------


    ¿Cuál es la velocidad típica de una conexión Ethernet en una red local?


    10/100/1000 Mbps (megabits por segundo)
    1/10/100 Gbps (gigabits por segundo)
    56 Kbps (kilobits por segundo)

    ---------------------------


    ¿Cuál de las siguientes opciones es una tecnología de red inalámbrica de corto alcance?


    Bluetooth
    Ethernet
    DSL

    ---------------------------


    ¿Qué es un switch de red?


    Un dispositivo que conecta múltiples dispositivos en una red y envía datos entre ellos
    Un tipo de cable de red
    Un software para bloquear sitios web

    ---------------------------


    ¿Qué es un servidor web?


    Un equipo dedicado a almacenar y entregar páginas web
    Un programa para editar imágenes
    Un dispositivo para escanear documentos

    ---------------------------


    ¿Qué es un proxy?


    Un servidor intermedio entre un cliente y un servidor de destino
    Una herramienta para realizar copias de seguridad de archivos
    Un programa para reproducir videos en línea

    ---------------------------


    ¿Cuál es el objetivo principal de un VPN (Virtual Private Network)?


    Crear una conexión segura y cifrada a través de internet
    Descargar software gratuito
    Reparar hardware dañado

    ---------------------------


    ¿Qué es un NAS (Network Attached Storage)?


    Un dispositivo de almacenamiento conectado a una red
    Un programa de edición de video
    Una forma de conectar impresoras en red

    ---------------------------


    ¿Qué es un cable Ethernet?


    Un cable utilizado para conectar dispositivos en una red
    Un tipo de cable para cargar dispositivos móviles
    Un cable utilizado para transmitir señales de televisión

    ---------------------------


    ¿Cuál es la función principal de un modem?


    Convertir señales digitales en señales analógicas y viceversa
    Realizar copias de seguridad de archivos
    Escanear documentos

    ---------------------------


    ¿Qué es un estándar IEEE 802.11?


    Un estándar para redes inalámbricas (Wi-Fi)
    Un protocolo de seguridad en internet
    Un tipo de impresora

    ---------------------------
